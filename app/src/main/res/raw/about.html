aMetro

This is a program for navigating in cities undergrounds and other public transport.

It displays transports schemes, allows you to find the shortest path between stations, and also describes what is next to the stations.

Open source components

  • Maps Icons Collection mapicons.mapsmarker.com
  • Flag Icons by Steven Skelton, github.com/stevenrskelton/flag-icon
  • Android-SVG, github.com/japgolly/svg-android

Maps source

  • pMetro project, pmetro.su
